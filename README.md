Este projeto é um programa interativo desenvolvido em Python que simula o controle de uma TV, 
permitindo ao usuário realizar ações como ligar/desligar, aumentar/diminuir o volume e visualizar o status atual da TV.
O código foi projetado para ser simples e direto, proporcionando uma experiência prática e divertida de controle.

Funcionalidades:

Ligar/Desligar TV: Permite ao usuário ligar ou desligar a TV, alterando o estado da TV.
Aumentar Volume: Aumenta o volume da TV em um valor fixo.
Diminuir Volume: Diminui o volume da TV em um valor fixo.
Exibir Status: Mostra se a TV está ligada ou desligada e o nível atual do volume.
Sair: Encerra o programa.
